# DagreSharp

A C# port of DagreJs (https://github.com/dagrejs/dagre) I have used for a personal project.

This initial commit is more or less a direct port of the javascript source. All unit test from the original project is adopted as well.
For now I have no detailed plans for this project, but may want to re-organize and/or rewrite parts of the code in the future.
